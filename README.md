# Placement Interaction System

A comprehensive platform for managing student placements.

## Components

### Server (Backend)
- **Framework**: Express.js
- **Database**: MongoDB
- **Auth**: JWT
- **Port**: 5000

### Client (Frontend)
- **Framework**: React (Vite)
- **Port**: 5173 (default Vite)

## Role-Based Functionality
- **Student**: Apply for jobs, track status.
- **Employer**: Post jobs, review candidates.
- **Placement Officer**: Track records, generate reports.
- **Admin**: All-access management.
